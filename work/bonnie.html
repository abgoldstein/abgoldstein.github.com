the elevator pitch

Bonnie is connected to the Cypress project and is a collaboration with popHealth. The project aims to allow for post-processing of HQMF documents, which are XML files that describe Clinical Quality Measures (CQM). There is a drag-and-drop UI for tweaking measure logic and the application can be used to export a handful of artifacts, including CQMs in a JSON format with generated executable JavaScript to calculate CQMs via MapReduce.

the cool parts i personally did

Primarily, I'm focused on making a unit test patient generator. Other members of the team who are parsing HQMF documents are able to give me a Ruby object representation of CQMs, and my goal is to create representations of patients that would exercise the logic of the measure. Not only can test patients serve to verify that our generated JavaScript is correct, but they can also assist EHR Vendors in their own testing without needing to worry about protected health information. I've also been working on packaging all of the Bonnie artifacts to import into Cypress serve as history for the evolution of measures and test patients.
